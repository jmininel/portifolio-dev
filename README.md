# 💼 Developer Portfolio

Um portfólio moderno e responsivo desenvolvido para apresentar minha trajetória como desenvolvedora Front-end, destacando projetos, habilidades técnicas e formas de contato.

## ✨ Destaques

* Interface moderna e responsiva
* Design focado em experiência do usuário
* Seção de projetos com informações detalhadas
* Tecnologias e habilidades organizadas
* Links para GitHub, LinkedIn e contato

## 🚀 Tecnologias

* Next.js
* React
* TypeScript
* Tailwind CSS
* Framer Motion
* React Icons

## ⚙️ Executando o projeto

```bash
# Clone o repositório
git clone https://github.com/jmininel/portifolio-dev.git

# Acesse a pasta
cd portifolio-dev

# Instale as dependências
pnpm install

# Execute o projeto
pnpm run dev
```

A aplicação estará disponível em:

```text
http://localhost:3000
```

## 🎯 Objetivo

Este projeto foi desenvolvido para reunir meus principais trabalhos em um único lugar, demonstrando minhas habilidades em desenvolvimento Front-end, boas práticas de código e criação de interfaces modernas.
